## About

**Ethereal** is a beautiful, fully responsive HTML5 template featuring a clean multi-panel layout with smooth scrolling, a photo gallery with lightbox support, and elegant typography. Ideal for portfolios, creative agencies, or personal showcase sites.

## Features

- Multi-panel full-screen layout
- Responsive design (mobile-friendly)
- Photo gallery with lightbox
- Smooth scroll navigation
- FontAwesome icons
- SASS source files included

## Tech Stack

- HTML5, CSS3 (compiled from SASS)
- jQuery, FontAwesome
